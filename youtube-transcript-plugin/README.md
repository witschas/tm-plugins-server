# YouTube Transcript Plugin for TypingMind

Extract transcripts (subtitles) from YouTube videos.

## Features

- Extract video ID from YouTube URLs
- Download full transcript
- Return transcript as plain text

## Usage Example

> "Can you get the transcript of the video https://www.youtube.com/watch?v=dQw4w9WgXcQ?"

## Import URL

```
https://github.com/witschas/tm-plugins-server/tree/main/youtube-transcript-plugin
```

## Requirements

- A plugin server must be set up
- Works with any hosting provider (Koyeb, Render, Railway, AWS, etc.)

## Notes

- Only works with videos that have available subtitles
- Some videos may not have transcripts available

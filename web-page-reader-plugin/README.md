# Web Page Reader Plugin for TypingMind

Read and extract text content from web pages automatically.

## Features

- Fetch HTML content from web pages
- Extract readable text content
- Remove ads, navigation, and unnecessary elements

## Use Cases

- Summarize articles and blog posts
- Analyze documentation
- Process web page content for AI analysis

## Usage Example

> "Read the article at https://example.com/article and summarize it."

## Import URL

```
https://github.com/witschas/tm-plugins-server/tree/main/web-page-reader-plugin
```

## Requirements

- A plugin server must be set up
- Works with any hosting provider (Koyeb, Render, Railway, AWS, etc.)

## Notes

- Only works with publicly accessible web pages
- Some websites may block automated requests
